Topgrad Notes

Design notes for a one-file autodiff library built on NumPy.

Goal

Single-file autodiff (`tensor.py`) wrapping NumPy arrays as tensors, with forward/backward for core NN ops and attention.

Ops

Softmax, ReLU, reshape, linear (matmul), conv via im2col, batch norm, layer norm, add, attention.

Optim

Optimizers live in `optim.py` (SGD and friends).

Examples & Tests

Examples under `examples/`; unit tests compare against PyTorch at tight tolerances.
